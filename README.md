# EPM Quant Reports

This repository hosts the daily Quantitative Reports for EPM Financial, created and delivered using automated forecasting and analysis tools.

Each report includes:
- 3-year performance charts with key technical indicators (20/50/200 MAs, resistance levels)
- Portfolio metrics including Sharpe Ratio, Alpha, Beta, ROE, and P/E
- Model forecasts using:
  - Linear Regression
  - Fama-French 3-Factor
  - Institutional (AQR-style) cross-sectional factors
  - QuantConnect-style Z-scoring
  - Gradient Boosted Machine Learning
- Sentiment analysis sourced from financial news APIs
- Plain-English commentary and forecasts per ticker
